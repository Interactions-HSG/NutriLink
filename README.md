# foodcoach-ontology

This is a repo for storing the ontology for FoodCoach-relevant studies. 

It mainly composes of three parts:

1. the digital receipts information
2. the food composition information from EatFit database https://eatfit-service.foodcoa.ch/
3. the information, e.g., demographics of users.

In the future, survey results might be added to this ontology as well.
